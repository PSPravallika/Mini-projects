# Snake Game (Python)

A classic Snake game implemented in Python, offering a simple and engaging gameplay experience.

## Features

- **Classic Gameplay:** Experience the traditional Snake game mechanics.
- **Arrow Key Controls:** Intuitive movement using keyboard arrow keys.
- **Score Tracking:** Keep track of your high score as you play.
- **Collision Detection:** Game ends upon collision with walls or the snake's own body.

## How to Play

1.  **Run the Game:** Execute the `snake_game.py` file (or your main game file) using a Python interpreter.
    ```bash
    python snake_game.py
